# steam-price-converter

一个 Tampermonkey 插件，能将 steam 商店和市场的的价格转换为某种货币展示。理论上支持所有区域。

## 使用

### 安装与更新

脚本托管于以下平台：

- [Script Cat](https://scriptcat.org/zh-CN/script-show-page/4099)
- [Greasy Fork](https://greasyfork.org/zh-CN/scripts/452504-steam%E4%BB%B7%E6%A0%BC%E8%BD%AC%E6%8D%A2)

> [!IMPORTANT]
> 由于 [steam++ 对 GM API 的支持有限](https://github.com/BeyondDimension/SteamTools/issues/2497)，该脚本目前**无法直接导入 steam++ (Watt Toolkit)**。

### 设置

![设置界面](https://s3.bmp.ovh/2026/02/24/aWCcBCyl.jpg)

#### 配置项详情

| 配置项             | 说明                                                 | 默认值  |
| :----------------- | :--------------------------------------------------- | :------ |
| **目标国家代码**   | 转换后的目标区域 (如 CN, US, HK, RU)                 | `CN`    |
| **目标货币符号**   | 显示在价格旁的货币符号 (如 ￥, $, HK$)                | `￥`     |
| **符号位置**       | 货币符号是否显示在数值之前                           | `是`    |
| **汇率缓存时间**   | 自动从接口更新汇率的间隔时间                         | `1小时` |
| **使用自定义汇率** | 是否手动指定汇率，开启后不再自动获取最新汇率         | `否`    |
| **自定义汇率**     | 手动指定的固定汇率数值                               | `1`     |
| **日志级别**       | 插件运行时的日志输出详细程度 (debug/info/warn/error) | `info`  |
| **市场页面禁用**   | 在 Steam 市场页面禁用插件 (防止与其他插件冲突)       | `否`    |

## 全球 Steam 价格格式参考

### 调研方法
本项目的数据来源主要分为两部分：

1. **货币定义与列表**：参考 [Steamworks 官方文档 - Supported Currencies](https://partner.steamgames.com/doc/store/pricing/currencies) 获取活跃的 ECurrency ID 及其对应的 ISO 4217 代码。
2. **格式实测**：通过访问 Steam 官方搜索页面并强制指定区域代码（Country Code）获取真实的 UI 渲染效果：`https://store.steampowered.com/search/?sort_by=Price_DESC&cc=[region_code]`

通过这种方式，我们将官方提供的技术规范与实际前端的显示习惯相结合，确保了价格解析的准确性。
